# 🏠 Airbnb Clone Project

에어비앤비(Airbnb) 웹 서비스를 클론 코딩한 프로젝트입니다.
숙소를 조회하고 상세 정보를 확인할 수 있으며, 사용자 경험을 고려한 UI와 데이터 처리를 구현하는 것을 목표로 진행했습니다.


## 📌 프로젝트 소개

Airbnb의 숙소 검색 및 상세 페이지 기능을 중심으로 구현한 클론 코딩 프로젝트입니다.
사용자가 숙소 정보를 확인하고 인터랙션을 경험할 수 있도록 UI와 상태 관리를 구현했습니다.


## 🛠 사용 기술 (Tech Stack)
- HTML
- CSS
- JavaScript


## ✨ 주요 기능 (Features)
1️⃣ 공통
- '언어와 지역'과 '통화'를 선택할 수 있습니다.

2️⃣ 메인 페이지
- 숙소 이미지, 가격 정보 등 등록된 숙소 목록을 확인할 수 있습니다.
